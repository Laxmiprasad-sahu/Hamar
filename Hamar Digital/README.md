# Hamar Digital Website

A premium full-stack website for the digital marketing agency *Hamar Digital* built with Next.js, Tailwind CSS, React, Framer Motion, GSAP, Node.js/Express and MongoDB.

## Features

- Responsive design across desktop, tablet, and mobile
- Smooth gradients, glassmorphism cards, and micro-animations
- Hero section, About, Story, Services, Portfolio, Reviews, Pricing, Contact
- Dedicated service pages with WhatsApp integration
- Contact form with backend API and MongoDB storage
- Express custom server with Next.js
- SEO-optimized, accessible, and fast

## Development Setup

1. Copy `.env.example` to `.env` and update with your MongoDB URI and port.
2. Install dependencies:
   ```bash
   npm install
   # or yarn install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Building

```bash
npm run build
npm start
```

The application uses a custom Express server defined in `server/index.js`. The server also proxies the contact API to the Next.js API route.

## Assets

Place the provided illustration files in `public/illustrations` and portfolio images in `public/portfolio`.

## Notes

- Ensure you have Node.js installed (>=14).
